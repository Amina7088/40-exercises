Бодлогын өгүүлбэр:

Танд n ширхэг бүхэл тоонууд бүхий массив болон нэг бүхэл тоо k өгөгдөнө. Массив дахь хэдэн (a, b) хосууд |a - b| = k нөхцөлийг хангаж байгааг ол.

Өөрөөр хэлбэл, хоёр элементийн ялгавар яг k байх хосын тоог ол.
