Танд n болон k гэсэн хоёр бүхэл тоо өгөгдөнө. Та 1-ээс n хүртэл бүхэл тоонуудын нэг пермутаци (давтагдалгүй эрэмбэ) ол. Пермутаци P нь дараах нөхцөлийг хангах ёстой:

∀i ∈ [1, n], |P[i] - i| == k
